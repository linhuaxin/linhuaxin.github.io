一、yum 安装：

编辑 Mongodb 安装源

编辑内容如下

[mongodb-org-3.6]
name=MongoDB Repository
baseurl=https://repo.mongodb.org/yum/redhat/$releasever/mongodb-org/3.6/x86_64/
gpgcheck=1
enabled=1
gpgkey=https://www.mongodb.org/static/pgp/server-3.6.asc

安装

修改配置文件的 bindIp, 默认是 127.0.0.1 只限于本机连接

启动命令
service mongod start

停止命令:
service mongod stop

重启命令:
service mongod stop

远程连接:
mongo linhuaxin.com:27017

添加一个用户：
db.createUser({user:’root’,pwd:’123456’,roles: [{role:’readWrite’,db:’app’}]})

修改密码:
use admin
db.changeUserPassword(‘root’,’123456’)
db.auth(‘root’, ‘123456’)

打开权限认证：
vim /etc/mongod.conf
security:
authorization: enabled
javascriptEnabled: false
